# Caluma Renovate Config Presets

Caluma presets for [Renovate](https://github.com/singapore/renovate) tool.

## Usage

Add this into `renovate.json`:

```json
{
  "extends": [
    "github>projectcaluma/renovate-config"
  ]
}
```
